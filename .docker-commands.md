# Docker WordPress Commands

## Quick Access

**WordPress URL:** http://localhost:8080
**WordPress Admin:** http://localhost:8080/wp-admin

## Useful Commands

### View Logs
```bash
docker-compose logs -f wordpress
```

### View Debug Logs
```bash
docker-compose exec wordpress tail -f /var/www/html/wp-content/debug.log
```

### Access WordPress Shell
```bash
docker-compose exec wordpress bash
```

### Stop WordPress
```bash
docker-compose down
```

### Start WordPress
```bash
docker-compose up -d
```

### Restart WordPress
```bash
docker-compose restart wordpress
```

### Check Plugin Files
```bash
docker-compose exec wordpress ls -la /var/www/html/wp-content/plugins/lunpetshop-chatbot/
```

### View Generated Image URLs
```bash
docker-compose exec wordpress grep -r "KittyCat Chatbot" /var/www/html/wp-content/debug.log
```

## First Time Setup

1. Visit http://localhost:8080
2. Complete WordPress installation (choose language, set admin credentials)
3. Go to Plugins → Activate "LùnPetShop KittyCat Chatbot"
4. Go to Settings → KittyCat Chatbot → Set API Base URL (e.g., `http://host.docker.internal:8000` or your tunnel URL)

## Testing the Plugin

After activation:
- Visit http://localhost:8080 (frontend)
- Look for the chat widget in bottom-right corner
- Open browser DevTools (F12) → Console to see debug messages
- Check for image loading errors
